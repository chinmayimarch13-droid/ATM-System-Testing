# ATM-System-Testing

## Project Overview

This project focuses on Manual Testing and Automation Testing concepts for an ATM System.

## Modules Tested

* PIN Validation
* Cash Withdrawal
* Balance Enquiry
* Mini Statement
* Invalid PIN Handling

## Manual Testing Activities

* Test Case Writing
* Positive Testing
* Negative Testing
* Boundary Value Analysis

## Test Scenarios

### PIN Validation

* Valid 4-digit PIN
* Invalid PIN
* Empty PIN
* Less than 4 digits
* More than 4 digits

### Cash Withdrawal

* Sufficient Balance
* Insufficient Balance
* Zero Amount
* Negative Amount

### Balance Enquiry

* Check Available Balance
* Verify Updated Balance

### Mini Statement

* View Recent Transactions
* Verify Transaction Details

## Skills Practiced

* Manual Testing
* Test Case Design
* Bug Reporting
* Boundary Value Analysis
* Equivalence Partitioning
